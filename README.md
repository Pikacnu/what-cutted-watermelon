# what-cutted-watermelon

## Installation

To install dependencies, run:

```bash
bun install
```

## Usage

To start the project, run:

```bash
bun run index.ts
```

## About

This project was created using `bun init` with bun v1.2.0. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
